Trabalho de Pensamento Computacional!

Primeiro, entendemos a importância de hospedar um site, permitindo que qualquer pessoa na internet possa acessá-lo.

Depois, aprendemos a utilizar o GitHub Pages, um serviço gratuito que permite publicar páginas web diretamente do repositório.

Em seguida, configuramos nosso projeto no GitHub e ativamos a hospedagem, tornando o site acessível por um link público.

Por último, testamos o site publicado, garantindo que todas as funcionalidades estavam operando corretamente.
